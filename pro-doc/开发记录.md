## 常见问题解决方案

### 问题1: 前后端开发环境启动问题 [✓]

**问题描述：** 需要同时启动前端和后端服务进行开发

**解决方案：**
使用项目根目录下的`start-dev.sh`脚本一键启动开发环境，该脚本会：

1. 启动Flask后端服务
2. 启动Vue前端开发服务器
3. 自动处理进程终止和清理

**使用方法：**

```bash
cd /home/wx/code/ysj
./start-dev.sh
```
